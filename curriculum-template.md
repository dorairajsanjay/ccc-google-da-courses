## Course Outlines

### Tier 1 Courses (1 Unit Each, 48 Hours)

#### DA-101: Data Analytics Foundations
- **Units:** 1 | **Hours:** 48 (16 lecture, 32 lab)
- **Description:** Introduction to data analytics concepts, analytical thinking, and the data analytics lifecycle through the Google Data Analytics Professional Certificate.
- **SLOs:**
  1. Define key data analytics concepts and terminology
  2. Describe the role of a data analyst in business decision-making
  3. Complete Google Data Analytics Coursera courses 1 and 2
  4. Demonstrate foundational analytical skills through hands-on assignments
- **Materials:** Computer with internet, Coursera account (institutional), Google account
- **Grading:** Coursera Completion 60%, Skills Verification 30%, Reflection 10%

#### DA-102: Data Preparation and Integrity
- **Units:** 1 | **Hours:** 48 (16 lecture, 32 lab)
- **Description:** Data collection, preparation, and integrity principles through the Google Data Analytics Professional Certificate.
- **SLOs:**
  1. Explain data types, structures, and collection methods
  2. Apply data cleaning and quality assurance techniques
  3. Complete Google Data Analytics Coursera courses 3 and 4
  4. Demonstrate data preparation skills through hands-on assignments
- **Materials:** Computer with internet, Coursera account (institutional), Google account
- **Grading:** Coursera Completion 60%, Skills Verification 30%, Reflection 10%

#### DA-103: Data Processing with SQL

#### DA-104: Data Visualization and Storytelling

### Tier 2 Courses (3 Units Each, 7-10 Weeks)

#### DA-201: Business Data Analytics
- **Units:** 3 | **Hours:** 144 (48 lecture, 96 lab)
- **Description:** Business context for data analytics including analytical thinking, spreadsheets, visualization, and stakeholder communication.
- **SLOs:**
  1. Apply analytical thinking frameworks to business problems
  2. Use spreadsheets for data organization and analysis
  3. Create effective data visualizations using Tableau
  4. Communicate data insights to business stakeholders
  5. Complete Google Data Analytics Coursera courses 1, 2, and 6
- **Materials:** Computer with internet, Coursera account, Google account, Tableau Public (free)
- **Grading:** Coursera 25%, Assignments 30%, Quizzes 10%, Discussions 5%, Final Project 30%

#### DA-202: SQL Data Analytics
- **Units:** 3 | **Hours:** 144 (48 lecture, 96 lab)
- **Description:** Advanced SQL for data retrieval, manipulation, and analysis using real-world datasets.
- **SLOs:**
  1. Write SQL queries to retrieve and filter data from relational databases
  2. Use aggregate and scalar functions for data analysis
  3. Combine data from multiple tables using joins and subqueries
  4. Apply normalization principles to database design
  5. Clean and prepare data using SQL techniques
  6. Complete Google Data Analytics Coursera courses 3, 4, and 5
- **Materials:** Computer with internet, Coursera account, Google BigQuery or local SQL environment
- **Grading:** Coursera 20%, SQL Assignments 35%, Quizzes 10%, Discussions 5%, Final Project 30%

#### DA-203: Data Visualization and Communication
- **Units:** 3 | **Hours:** 144 (48 lecture, 96 lab)
- **Description:** Theory and practice of data visualization, Tableau mastery, data storytelling, and presentation skills.
- **SLOs:**
  1. Apply visualization theory and perceptual principles to chart design
  2. Select appropriate chart types for different data and audiences
  3. Build interactive dashboards in Tableau
  4. Use LOD expressions and parameters for advanced Tableau analysis
  5. Construct data narratives and deliver effective presentations
  6. Complete Google Data Analytics Coursera course 6
- **Materials:** Computer with internet, Coursera account, Tableau Public (free), Google account
- **Grading:** Coursera 20%, Assignments 30%, Quizzes 10%, Discussions 5%, Final Portfolio 35%

#### DA-204: Python for Data Science
- **Units:** 3 | **Hours:** 144 (48 lecture, 96 lab)
- **Description:** Python programming for data analysis including pandas, matplotlib, and exploratory data analysis.
- **SLOs:**
  1. Write Python code using variables, control flow, and functions
  2. Work with Python data structures (lists, dictionaries, tuples)
  3. Use pandas for data manipulation, cleaning, and analysis
  4. Create data visualizations using matplotlib and seaborn
  5. Conduct exploratory data analysis workflows
  6. Complete Google Data Analytics Coursera course 7
- **Materials:** Computer with internet, Coursera account, Google Colab or local Python environment
- **Grading:** Coursera 20%, Coding Assignments 25%, Quizzes 10%, Discussions 5%, Midterm 15%, Final 25%

#### DA-205: Data Analytics Capstone
- **Units:** 3 | **Hours:** 144 (48 lecture, 96 lab)
- **Description:** End-to-end analytics capstone project from dataset selection through presentation and portfolio.
- **SLOs:**
  1. Define research questions and scope a feasible analytics project
  2. Collect, clean, and prepare data for analysis
  3. Conduct exploratory data analysis and apply analytical techniques
  4. Build an interactive Tableau dashboard
  5. Present findings through a structured data narrative
  6. Assemble a professional analytics portfolio
  7. Provide constructive peer feedback on classmates' projects
- **Materials:** Computer with internet, Google Colab or Python environment, Tableau Public, Google account
- **Grading:** Proposal 10%, Data Prep 15%, Analysis 15%, Dashboard 20%, Presentation 20%, Portfolio 10%, Peer Reviews 10%

## BLS Labor Market Data Justification

**Occupation:** Data Analyst / Data Scientist
**SOC Codes:** 15-2051 (Data Scientists), 15-1221 (Computer and Information Analysts)

- **Median Annual Wage (2024):** $108,020 (Data Scientists), $102,240 (Computer and Information Analysts)
- **Projected Job Growth (2022-2032):** 35% (Data Scientists) — much faster than average
- **Annual Job Openings:** Approximately 17,700 (Data Scientists)
- **California Employment:** California is the largest employer of data analysts and data scientists in the nation

**Regional Demand (Bay Area / California):**
- Silicon Valley, San Francisco, and Los Angeles are top metro areas for data analytics positions
- California Community College students have direct access to the state's largest technology employment market
- Entry-level data analyst positions in California typically require a certificate or associate degree plus demonstrated portfolio skills

**Industry Alignment:**
- The Google Data Analytics Professional Certificate is one of the most recognized industry credentials
- Coursera reports that 75% of certificate completers report positive career outcomes within 6 months
- The CCC pathway adds college credit and structured instructor support to the industry credential
