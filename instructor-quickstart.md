# Instructor Quick-Start Guide
## CCC Google Data Analytics Certificate Courses

---

## Step 1: Import IMSCC Files into Canvas

1. Log in to Canvas as an instructor
2. Navigate to your course shell
3. Go to **Settings → Import Course Content**
4. Select **Canvas Course Export Package** as the content type
5. Choose the `.imscc` file for your course (e.g., `DA-201.imscc`)
6. Click **Import** and wait for the process to complete
7. Check the import log for any warnings

**After import:**
- All modules will be in "unpublished" state — publish them as you are ready
- Review the Instructor Resources module (the last module) — it is unpublished
  and hidden from students by default
- Check that all wiki pages appear in the correct modules

---

## Step 2: Set Up Coursera Institutional Access

1. Contact your institution's Coursera administrator to get access to the
   Coursera for Campus (or Coursera for Government) program
2. Request that the Google Data Analytics Professional Certificate courses
   be added to your institutional program
3. Get the enrollment link or invitation code for students
4. Test the enrollment process yourself before sharing with students

**If your institution does not have Coursera access:**
Contact the CCC Data Analytics program coordinator for assistance with
institutional licensing.

---

## Step 3: Enroll Students in Coursera Programs

1. Share the enrollment link with students through Canvas Announcements
2. Set a deadline for Coursera enrollment (recommend: end of Week 1)
3. Use the Coursera admin dashboard to verify student enrollment
4. Follow up with students who have not enrolled by the deadline

---

## Step 4: Set Due Dates

1. Review the **Instructor Guide: Recommended Due Dates** page in the
   Instructor Resources module
2. In Canvas, go to each assignment/quiz/discussion and set due dates
3. Use a consistent weekly pattern (e.g., Sundays at 11:59 PM)
4. Consider your institution's academic calendar and holidays
5. Publish the course schedule in Canvas Announcements

**Tip:** You can use Canvas's "Assign" feature to set due dates for multiple
items at once from the Assignments page.

---

## Step 5: Handle Student Certificate Submissions

### Coursera Certificates
- Students submit screenshots or PDFs of their Coursera completion page
- Verify that the student's name and correct course title are visible
- Grade on a complete/incomplete basis

### College Certificates
- Direct students to their college advisor for information about earning
  a college certificate or credential
- The college certificate is separate from the Coursera certificate
- Requirements vary by institution

---

## Step 6: Use the Instructor Resources Module

The last module in every course is **Instructor Resources**. It is unpublished
(invisible to students) and contains:

| Page | What It Covers |
|------|----------------|
| Recommended Due Dates | Suggested timeline for all assignments |
| Coursera Setup | How to manage Coursera institutional access |
| Assessments and Grading | Tips for grading each assignment type |
| Student Certificates | The two certificates and what instructors do |
| Support Resources | Links to help resources and instructor community |

## Common Questions

**Q: Do I need to create the Coursera courses?**
No. The Google Data Analytics Professional Certificate already exists on Coursera.
You just need institutional access so students can enroll for free.

**Q: Can I modify the course content?**
Yes. After importing, you can edit any page, assignment, quiz, or discussion in
Canvas. The IMSCC file is a starting point.

**Q: What if a student already has a Coursera certificate?**
They can submit their existing certificate for the Coursera completion assignments.
They still need to complete the college-specific assignments (skills verification,
quizzes, discussions, projects).

**Q: How do CPL courses work?**
CPL (Credit for Prior Learning) courses are separate IMSCC packages with only
an orientation module and skills verification assessments. See the CPL Guidelines
document for details.
